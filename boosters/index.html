Boosters e Produtos

Guia completo dos produtos de Sorcery: Contested Realm disponiveis. Sets, boosters, precons e mais.

Tipos de Produto

Booster Pack

Pacote individual com cartas aleatorias. Ideal para abrir e colecionar.

12 cartas por pack, incluindo 1 rara ou superior garantida

Booster Box

Caixa com 36 boosters. Melhor custo-beneficio para colecionar um set completo.

36 packs, aproximadamente 432 cartas

Precon Deck

Deck pre-construido pronto para jogar. Otimo para iniciantes.

Deck completo com 50 cartas + 30 sites + avatar

Starter Kit

Kit para dois jogadores com dois decks e materiais de aprendizado.

2 decks, regras, playmat de papel

Sets Disponiveis

Alpha & Beta Fundacao

Os sets originais que estabeleceram o mundo de Sorcery. Beta e uma reimpressao de Alpha com pequenas correcoes. Cartas fundamentais para qualquer colecao.

Arthurian Legends Expansao

Primeira expansao tematica, trazendo as lendas do Rei Arthur para o Realm. Cavaleiros, Camelot, Excalibur e a Tavola Redonda.

Gothic Mais Recente

Set com tematica de horror gotico. Vampiros, lobisomens, e criaturas das trevas. Introduz novos avatares como o Necromancer.

Dragonlord Proximo

Proximo set anunciado, focado em dragoes e novas mecanicas elementais. Lancamento previsto para 2026.

Dicas de Compra

Para iniciantes: Comece com um Starter Kit ou Precon Deck. Voce tera um deck pronto para jogar e aprender as regras.

Para colecionar: Booster Boxes oferecem o melhor custo por carta. Comprar boxes de sets diferentes diversifica sua colecao.

Para competir: Compre singles (cartas avulsas) nas lojas brasileiras. E mais eficiente do que abrir boosters buscando cartas especificas.
